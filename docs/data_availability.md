# Data Availability

This repository contains code and non-sensitive gene-level metadata only. It does not include raw patient-level expression matrices, methylation matrices, clinical files, controlled-access survival data, DepMap/GDSC matrices, or final high-resolution figure binaries.

## Expected Local Input Structure

The scripts expect a local `EOBC_BIOMARKER_ROOT` directory with the following major subfolders:

- `00_inputs_detected/`
  - `TPM_young.csv`
  - `MET_young_batch_JW.csv`
  - `total_sample_clinical_all.csv`
- `09_selected_relaxed_union_family6_final_signedMeth/tables/`
  - `selected_union_biomarker_genes_family6_v18.csv`
- `13_final_manuscript_validation_suite_v5_full_rewrite_raw_depmap_SVS_MAF/tables/`
  - DepMap/GDSC drug association tables
  - dose-response curve tables
  - immune TIL/TMB correlation tables
- `05_model_output_archive/biomarker_gene_moment_RF_SEM_manual_groups_v2/tables/`
  - `gene_driver_summary_all_targets.csv`
- `final_analysis/`
  - intermediate and final analysis outputs generated by the workflow scripts

Use `config/paths_template.yml` and `.env.example` to configure equivalent local paths.

## External Resources

The workflow uses public or controlled-access resources that must be obtained according to their respective data-use terms:

- TCGA breast cancer expression, methylation, clinical, immune, and mutation/TMB-derived resources.
- DepMap/GDSC drug-response resources for AUC/IC50 and dose-response analyses.
- Internally curated EOBC marker-panel and model-output tables generated by the analysis workflow.

## Privacy and Sharing Boundary

Only derived, gene-level verification tables are included in `metadata/`. These tables are intended to document figure consistency and do not contain patient identifiers or sample-level measurements.
