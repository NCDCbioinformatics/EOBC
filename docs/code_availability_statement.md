# Code Availability Statement

All code used to generate the EOBC TSS-methylation biomarker figures, survival/drug/immune evidence summaries, and final integrated Sankey visualization is available at:

`https://github.com/NCDCbioinformatics/EOBC`

The repository includes R and Python scripts, a reproducibility workflow, path templates, and gene-level figure-audit metadata. Raw patient-level data, controlled-access clinical/molecular matrices, and bulky generated figure outputs are not distributed in the repository and should be obtained through the appropriate data-access mechanisms described in the manuscript.
