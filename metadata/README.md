# Metadata Files

These files are non-sensitive, gene-level verification outputs copied from the final manuscript figure run.

- `publication_final_figure_manifest.csv`: final figure-panel manifest and interpretation notes.
- `meth_group_pathway_evidence_biomarker_right_sankey_final_audit_v12.csv`: final Sankey source-audit table verifying EOBC group, Meth-RNA label, RF layer scores, and supplement agreement.
- `supplement_vs_sankey_evidence_reconciliation.csv`: final evidence-route reconciliation between individual supplement panels and Sankey routes.

The full raw and intermediate tables used to generate these metadata files are not committed because they may contain sample-level or controlled-access information.
